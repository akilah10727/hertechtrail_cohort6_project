Installation

To install and run this portfolio locally, follow these steps:

Clone the repository: git clone https://github.com/akilah10727/hertechtrail_cohort6_project.git ( Initially forked from https://github.com/PrincessMaggy/hertechtrail_cohort6_project.git: )
          
Navigate to the project directory: cd portfolio
Open the index.html file in your preferred web browser.
Usage

Feel free to explore the different sections of the portfolio to learn more about my projects. The main navigation menu provides links to the Home, About, Portfolio, and Contact pages.

On the Portfolio page, you can find a collection of my works. Each project is displayed as a thumbnail image with a title and category. Clicking on a project will open a modal window with additional details, including screenshots and links to view the project's live demo and source code.
